# Code of Conduct

Aura is a small project. The rules are simple.

## Be decent

- Assume good faith. Most people are trying to help.
- Disagree about ideas, not about people. Code review is about the code.
- Keep it on-topic. Issues and PRs are for the project, not for unrelated arguments.
- Harassment, personal attacks, slurs, or sustained disruption are not welcome here, regardless of the target.

That's it. If you wouldn't say it to a colleague's face in a professional setting, don't post it here.

## If something's wrong

If someone's behavior in this project is making it hard for you to participate, email **jon@gnupg.net**. Reports are read by the maintainer and not shared further unless action requires it.

Responses range from a private word to removal from the project, depending on what happened. The maintainer's call is final for this project; if you disagree, you're free to fork.

## Scope

This applies in project spaces — the GitHub repo, issues, PRs, discussions, and any official chat channels — and to behavior elsewhere that materially affects the project or its participants.
